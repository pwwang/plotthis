# Import a font

Imports a font for use in plots. This function imports a font from the
`plotthis` package, system fonts, or Google fonts. This is borrowed from
the `plummy` package. To check available fonts, use
[`list_fonts()`](https://pwwang.github.io/plotthis/reference/list_fonts.md).

## Usage

``` r
import_font(font)
```

## Arguments

- font:

  Font family name to import.

## Value

Invisible `TRUE`.
